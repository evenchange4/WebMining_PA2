## Run

* 查看bash 位置

```
$ which bash
=> /bin/bash
```

* 調整權限

```
$ chmod +x executeLR.sh
$ chmod +x executePR.sh
```

* 執行程式 PageRank

```
$ ./executePR.sh cs.stanford.2004-10.graph
```

* 執行程式 LexRank
```
$ ./executePR.sh cs.stanford.2004-10.graph
```
